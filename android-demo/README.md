PeanutClip Android Demo APK

Wat is dit
- Dit is een simpele Android wrapper (WebView) rond de fake mobile demo.
- De demo-assets zitten lokaal in de app, dus hij werkt zonder internet.

Projectpad
- android-demo/

Snel APK bouwen in Android Studio
1. Open Android Studio.
2. Kies Open en selecteer de map android-demo.
3. Wacht op Gradle sync.
4. In menu: Build > Build Bundle(s) / APK(s) > Build APK(s).
5. Na build klik Locate en pak het APK-bestand uit app/build/outputs/apk/debug/.

APK installeren op je telefoon
1. Zet het APK-bestand op je telefoon (WhatsApp, Drive, USB, etc).
2. Open het bestand op je telefoon.
3. Sta Install unknown apps toe voor de app waarmee je installeert.
4. Installeer PeanutClip Demo.

Sneller direct installeren via USB
1. Zet Developer options + USB debugging aan op je telefoon.
2. Verbind je telefoon met USB.
3. In Android Studio klik Run app op je device.

Tip
- Wil je een app-icoon op maat, vervang dan icon assets via Android Studio: New > Image Asset.
