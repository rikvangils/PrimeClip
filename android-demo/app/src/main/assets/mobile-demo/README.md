PeanutClip Mobile Demo

Doel
- Dit is een fake mobiele demo om het concept visueel te tonen.
- Het is geen werkende productie app.

Snelle telefoon install via GitHub Pages
1. Push deze map naar GitHub op branch main.
2. Open op je telefoon de URL:
   https://rikvangils.github.io/PrimeClip/mobile-demo/
3. In Chrome Android: menu rechtsboven > Add to Home screen.
4. Geef een naam en bevestig.

Lokale test op Windows
1. Open een terminal in de projectmap.
2. Start een simpele server:
   python -m http.server 5500 -d docs
3. Open op je pc:
   http://127.0.0.1:5500/mobile-demo/

Lokale test op je telefoon in hetzelfde wifi netwerk
1. Zoek je lokale IP op je pc (bijv 192.168.1.20).
2. Open op je telefoon:
   http://192.168.1.20:5500/mobile-demo/
3. Voeg toe aan je homescreen via browsermenu.
