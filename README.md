## Over dit project

Dit is een blanco startpunt voor een nieuw idee, uitgewerkt via de **BMAD-workflow**.

## BMAD Workflow – Snelle gids

De BMAD-workflow helpt je stap voor stap van idee naar implementatie:

| Fase | Stap | Commando / actie |
|------|------|-----------------|
| 1 | Product Brief | _"lets create a product brief"_ |
| 2 | PRD (Product Requirements) | _"lets create a product requirements document"_ |
| 3 | UX Design | _"lets create UX design"_ |
| 4 | Architectuur | _"lets create architecture"_ |
| 5 | Epics & Stories | _"create the epics and stories list"_ |
| 6 | Sprint Planning | _"run sprint planning"_ |
| 7 | Story implementeren | _"dev this story [story file]"_ |

## Mapstructuur

```
BMAD-New-Idea/
├── docs/
│   └── bmad/
│       ├── product-brief.md      ← stap 1 (invullen)
│       ├── prd.md                ← stap 2 (aanmaken na brief)
│       ├── ux-design.md          ← stap 3
│       ├── architecture.md       ← stap 4
│       └── epics-and-stories.md  ← stap 5
├── src/                          ← broncode (later)
└── README.md
```

## Hoe te beginnen

1. Open deze map als workspace in VS Code
2. Zeg tegen GitHub Copilot: **"lets create a product brief"**
3. Beantwoord de vragen over jouw idee
4. Volg de BMAD-stappen in volgorde
